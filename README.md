# Spaced repetition app frontend

Made with React and Apollo client

## Installation

```bash
$ npm install
```

## Development

```bash
$ npm run start
```

## Test

```bash
# tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Generate graphql types

Running server is required (configuration is in codegen.yml)

```bash
$ npm run generate
```
